# NBA Daily Challenge Data

Auto-updated daily at 6:00 AM UTC via GitHub Actions.

Provides player data for the NBA Legacy daily challenge mode:
- Current season averages (2025-26)
- Recent game box scores (last 5 days)

## Data URL

```
https://oliverchristensen93.github.io/nba-daily-data/daily_challenge.json
```

## Manual trigger

Go to Actions > "Fetch NBA Daily Data" > Run workflow
